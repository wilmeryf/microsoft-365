# Introdução ao Excel

> **Data:** 22 de julho de 2026

---

## O que é?

O Excel é um software que permite criar e editar planilhas digitais.

### Planilha

Uma planilha é uma estrutura quadriculada que pode ser usada de diversas maneiras, vão de simples cálculos matemáticos até tabelas usadas por empresas.

As empresas utilizam para poderem se organizar, seja com tabela de controle de caixa, controle de estoque, controle de pagamentos, entre outras aplicações.

---

## História

No início da década de 1980, com o surgimento dos primeiros computadores pessoais, muitas pessoas ainda não enxergavam uma necessidade prática para adquiri-los.

Para demonstrar seu potencial, surgiram os primeiros editores de planilhas eletrônicas, considerados uma **"Killer App"**, ou seja, um software capaz de justificar a compra de um computador por si só.

Entre os programas mais populares da época destacavam-se:

- Multiplan;
- Lotus 1-2-3.

Observando o sucesso dessas soluções, a Microsoft lançou, em 1985, a primeira versão do Microsoft Excel, que ao longo dos anos evoluiu até se tornar um dos softwares de planilhas mais utilizados.

---

## Boas Práticas

É recomendável realizar cópias de segurança das planilhas periodicamente para evitar a perda de informações.

Os arquivos podem ser armazenados em diferentes locais, como:

- OneDrive;
- Pen drive;
- HD externo;
- SSD externo;
- Armazenamento em nuvem;
- Outro dispositivo confiável.
